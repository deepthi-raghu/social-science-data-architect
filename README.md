# Social Science Data Architect

### 📌 Overview
This repository showcases the analytical framework and data pipelines I developed while serving as a **Research Assistant** at **Indiana University Bloomington**. The project was designed to process large-scale social science datasets to uncover underlying patterns and contribute to peer-reviewed research.

### 🏗 Key Features
- **End-to-End Pipeline:** Automated cleaning, transformation, and normalization of heterogeneous data sources.
- **Statistical Modeling:** Implementation of advanced regressions and clustering to identify significant correlations in social behavior.
- **Impactful Insights:** Built to provide Professor Sarah Imhoff's research team with a scalable way to validate complex hypotheses.
- **Visualization:** Custom plotting modules for presenting high-dimensional data in an intuitive format.

### 🧬 Tech Stack
- **Core:** Python (Pandas, NumPy, Scipy).
- **Analysis:** Scikit-learn, Statsmodels.
- **Visualization:** Matplotlib, Seaborn.

### 💡 Reflection
While my current focus is on production-scale ML at Apple, this project remains the foundation of my approach: **rigorous data integrity leads to superior model performance.**
